Условия использования

Настоящие условия регулируют доступ и использование сервиса Habits. Используя приложение, вы подтверждаете, что ознакомились с этими условиями и принимаете их.

Последнее обновление: 1 апреля 2026

1. Общие положения

Habits — это сервис для ведения и отслеживания привычек. Пользователь может использовать приложение как в гостевом режиме, так и через зарегистрированный аккаунт.

Если вы не согласны с настоящими условиями, пожалуйста, не используйте сервис.

2. Функциональность сервиса

Сервис может предоставлять следующие возможности:

  • создание и редактирование привычек;
  • отметка выполнения привычек;
  • хранение истории действий;
  • синхронизация данных между устройствами после регистрации или входа;
  • удаление аккаунта и связанных данных.

3. Регистрация и аккаунт

Для доступа к синхронизации и облачному хранению данных пользователь может создать аккаунт, указав email и пароль.

Пользователь обязуется:

  • предоставлять корректный email;
  • обеспечивать конфиденциальность своих данных для входа;
  • не передавать доступ к аккаунту третьим лицам.

4. Гостевой режим

До регистрации пользователь может пользоваться приложением без создания аккаунта. В этом случае часть данных может храниться локально на устройстве пользователя.

После регистрации или входа локальные данные могут быть перенесены в аккаунт для синхронизации между устройствами.

5. Данные пользователя

Для работы сервиса могут использоваться:

  • данные аккаунта, включая email;
  • данные, созданные пользователем внутри приложения;
  • локально сохранённые данные, связанные с использованием сервиса;
  • технические данные, необходимые для поддержки сессии и корректной работы приложения.

6. Ограничения использования

Пользователь обязуется не использовать сервис для действий, которые нарушают применимое законодательство, мешают работе приложения или направлены на получение несанкционированного доступа к данным других пользователей.

7. Удаление аккаунта

Пользователь может удалить свой аккаунт через интерфейс приложения. При удалении аккаунта связанные с ним данные удаляются без возможности восстановления в рамках стандартного пользовательского интерфейса.

8. Ограничение ответственности

Сервис предоставляется по принципу «как есть». Мы прилагаем разумные усилия для обеспечения стабильной работы приложения, но не гарантируем абсолютное отсутствие ошибок, перебоев или технических ограничений.

9. Изменение условий

Настоящие условия могут быть изменены или дополнены. Актуальная версия публикуется на этой странице.

10. Контакты

Если у вас есть вопросы по условиям использования, свяжитесь с владельцем сервиса через контактный канал, указанный в приложении.
